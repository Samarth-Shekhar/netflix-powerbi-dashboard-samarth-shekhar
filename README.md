## About the Dataset

Netflix is one of the leading video streaming platforms with millions of users worldwide.  
The dataset used in this project contains information about movies and TV shows available on Netflix, including:

- Title
- Type (Movie / TV Show)
- Genre
- Country
- Release Year
- Rating
- Duration
- Director
- Cast

The dataset is in CSV format and is imported directly into Power BI Desktop.

## Usage

To explore the dashboard:

1. Download and install **Power BI Desktop**
2. Download or clone this repository
3. Open the `.pbix` file using Power BI Desktop
4. Interact with the visuals using filters and slicers
